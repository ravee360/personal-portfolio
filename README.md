# Portfolio Website

Personal portfolio website built with React, TypeScript, and Tailwind CSS.

## 🚀 Live Demo

[View Live Portfolio](https://your-portfolio.vercel.app) *(Update after deployment)*

## 🛠️ Tech Stack

- **React 18** - UI Library
- **TypeScript** - Type Safety
- **Vite** - Build Tool
- **Tailwind CSS** - Styling
- **Lucide React** - Icons

## 📦 Installation

```bash
npm install
```

## 🏃 Development

```bash
npm run dev
```

Visit `http://localhost:5173`

## 🏗️ Build

```bash
npm run build
```
